generate_coding_page_multiproc(list_of_records, num_procs, idm)

source code 

Using a pool of processes, generate coding pages for the target records in list_of_records. To do this, first fetch a list of similar records for each target record. If the ecord is one from the database, then it has an id and an assigned identity. In this case, we will display two sets of identities:

  1. identities that are linked to the target identity or "focal cluster".
  2. identities containing other records that were similar to our target record.

list_linked_identities is the list of all identities linked to focal_identity in identities_adjacency. list_similar_identities is the list of all identities containing records similar to the target_record but which aren't linked to it in identities_adjacency.

Parameters:
  • list_of_records - list of target records.
  • num_procs - number of processes to use.
  • idm - an IdentityManager instance.

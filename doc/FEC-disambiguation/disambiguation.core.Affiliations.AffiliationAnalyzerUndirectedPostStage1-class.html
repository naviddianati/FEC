Subclass of AffiliationAnalyzerUndirected. Generates a similar affiliation graph, but instead of the high-certainty matches found pre-stage1, it uses the stage1 identities to link affiliation identifiers. For this, we need to override the load_data() method so it loads the records using an FECretriever. Must set self.G, self.dict_record_nodes.

verdict(self, result)

source code 

Issue a verdict on an identity pair based on their detailed comparison results.

Parameters:
  • result - a list of three values. The values are result_name, result_occupation and result_employer Each one is a tuple. For the details of the result formats see Person.compare.
Returns:
-1, 0 or 1. -1 means there is a middle name conflict that could not be rule out as an error based on exact affiliation matches. 0 means not a match, but not a middle name conflict either. 1 means a match.
Overrides: VerdictAuthorityBase.verdict

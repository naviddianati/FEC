Class Town

source code

A wrapper around the persons objects. The main reason this is used is to allow the person.neighbor to contain integer id's of the neighbors rather than the person objects themselves. This solves an recursion problem when pickling D.set_of_persons. So, now the Town can receive a list of neighbor id's and return a list of the actual person objects themselves. The key feature allowing this is the dict_persons within the Town.

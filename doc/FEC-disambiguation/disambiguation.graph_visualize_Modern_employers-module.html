set_group_markers(G, membership_vector, palette)

source code 

Given a graph and a membership_vector (list), this functions computes a dictionary {(indices):color} that can be passed Graph.plot() 's mark_groups property. Also, a new edge "weight" property is added that vastly distinguishes inter-cluster edges from intra-cluster edges to assist in a community-respecting layout.

collapse_graph(G, identity_function, comparison_mode='vertex')

source code 

This function returns a collapsed version of G by contracting some edges. the function identity_function should be able to take two vertices as arguments and return True of False indicating whether the two should be considered identical.

Module Hierarchy

  • disambiguation
    • disambiguation.config: This module defines global variables used uniformly by various scripts.
    • disambiguation.core: This subpackage contains almost all of the important modules used for disambiguation.
      • disambiguation.core.Affiliations: This module implements classes used to generate the affiliation (employer/occupation) similarity graphs.
      • disambiguation.core.Database: This module defines the DatabaseManager class and its subclasses, used for interacting with the MySQL server for retrieving records from the state and national tables, writing to the identities table, retrieving the computed identities from the identities table, etc.
      • disambiguation.core.Disambiguator: This module defines the Disambiguator class which is the working engine of the disambiguation process.
      • disambiguation.core.Partitioner
      • disambiguation.core.Person: This module defines the Person class which groups records together, representing an identity.
      • disambiguation.core.Project: This mofule defines the Project class which can be used to encapsulate various administrative functions such as logging, exporting textual results, etc.
      • disambiguation.core.Record: The module defines the Record class which represents a record from the database and implements various methods, most importantly for record comparison.
      • disambiguation.core.Tokenizer: This module defines a number of classes used to parse and tokenize the data within records.
      • disambiguation.core.Town: This module defines the Town class which is a convenience container of Person objects.
      • disambiguation.core.address
      • disambiguation.core.filters: This module implements functions for computing the statistical significance of the weighted edges of the affiliation (employer/occupation) graphs using the Marginal Likelihood Filter of Dianati.
      • disambiguation.core.hashes: This module contains the worker function that computes the LSH hashes for the records.
      • disambiguation.core.memory_spike
      • disambiguation.core.mypalettes
      • disambiguation.core.search: This module provides methods for searching the FEC database using various types of queries.
      • disambiguation.core.states: This module contains data on US states, and methods that retrieve data on US states.
      • disambiguation.core.utils: This module loads commonly used packages and modules.
    • disambiguation.data: This module defines variables used for accessing data files used internally by various modules.
      • disambiguation.data.compileNames
      • disambiguation.data.compile_name_variants: This script reads rows of the file ../data/name-variants-raw.csv.
      • disambiguation.data.disp
    • disambiguation.graph_visualize
    • disambiguation.graph_visualize_Modern_employers
    • disambiguation.graph_visualize_newyork_occupations
    • disambiguation.init: This module contains methods that prepare data and export intermediate computation results to files.
    • disambiguation.memory_spike
    • disambiguation.nameparser
      • disambiguation.nameparser.constants
      • disambiguation.nameparser.parser
    • disambiguation.script_generate_affiliations
